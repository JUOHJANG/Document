## 0. 개요 (Overview)
MSTR Web 솔루션을 구현한다. 작업 영역은 아래와 같다.
 - 웹 소스 공통 모듈화 
 - 웹 커스터마이징
 - 포탈 화면 제공
 - D3 시각화 차트
